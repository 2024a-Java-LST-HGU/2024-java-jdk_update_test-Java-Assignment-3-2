# 課題 3-2: Switch文

### 課題の説明
キーボードから繰り返し整数を受けとり、数値に応じて下記を表示するプログラムを作成しなさい。
ただし、繰り返しは3回で終了させること。

|数値| 表示        |
|---|-----------|
|1| グー        |
|2| チョキ       |
|3| パー        |
|それ以外| 不適切な入力です  |
